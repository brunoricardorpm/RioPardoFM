# 📻 Rádio Rio Pardo MG - Android App

Este é o aplicativo oficial da **Rádio Rio Pardo MG**, desenvolvido para oferecer streaming de áudio de alta qualidade diretamente em dispositivos Android.

## 🚀 Versão Atual: 1.4 (Estável)
Esta versão contém correções críticas de estabilidade, incluindo:
- Proteção contra crashes na inicialização (WorkManager/ProGuard).
- Unificação das bibliotecas Media3 para melhor performance.
- Interface moderna construída com **Jetpack Compose**.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Engine de Áudio:** [Media3 / ExoPlayer](https://developer.android.com/guide/topics/media/media3)
- **Carregamento de Imagens:** [Coil](https://coil-kt.github.io/coil/)
- **Anúncios:** [Google AdMob](https://admob.google.com/)

## 📂 Estrutura do Projeto
- `app/`: Contém todo o código fonte do aplicativo Android.
- `app/src/main/java/br/com/britoinformatica/www/riopardofm/`: Lógica principal da rádio e UI.
- `app/proguard-rules.pro`: Regras de segurança para garantir o funcionamento na versão final da Play Store.
